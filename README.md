# ポーカーゲーム
## 概要
以下のポーカーのルールを実装
1. プレイヤーとディーラー(対戦相手)に5枚ずつ手札を配布
2. プレイヤーが手札を交換
3. ディーラーが配られた手札の種類に応じて手札を交換
4. お互いの手札を交換
<br>

## 使い方
1. ターミナル上で実行。`ruby main.rb`で実行
2. 画面の案内に従い、キー操作
<br>

## インストール
```
$ git clone git@github.com:Taiji-Namba/Poker.git
$ cd poker
$ ruby main.rb
```
